# 环境要求

  • nodejs >= v14
  • yarn
  • git
  • github

# 安装 nodejs

从 nodejs 官网open in new window 下载并安装最新版，然后使用下面的命令检查是否成功安装

node -v
npm -v
1
2

# 安装 yarn

在安装完 nodejs 之后，使用命令安装 yarn

npm i -g yarn
1

检查是否成功安装

yarn -v
1

# 注册 github

进入 github 官网open in new window，点击 sign up，输入邮箱和密码即可成功注册。

# 安装 git

从 git 官网open in new window 下载 git 安装包（如果是 linux/macos，那么你可能已经安装它了），安装完成后检查

git --version
1

然后设置你的本地用户名和邮箱

git config --global user.name "FIRST_NAME LAST_NAME"
git config --global user.email "MY_NAME@example.com"
1
2

具体如何结合 git 和 github 一起使用请参考 github 官方文档open in new window
