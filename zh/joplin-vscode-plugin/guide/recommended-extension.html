# 插件

先简单列出吾辈与笔记相关的插件列表，它们都实现了某项特定功能且可以组合。

  • Markdown All in Oneopen in new window: Markdown 语言支持
  • markdown lintopen in new window: Markdown linter 工具
  • Prettieropen in new window: 多语言格式化工具，对 markdown 的支持非常好
  • PicGoopen in new window: 图片上传（这是一个第三方程序）
  • Markdown PDFopen in new window: Markdown 导出 PDF
  • Draw.io Integrationopen in new window: draw.io 流程图集成
    • Markdown Preview Mermaid Supportopen in new window: 支持 mermaid 流程图 DSL，但 draw.io 更好用
  • vscode-mindmapopen in new window: 思维导图集成（此为 fork 版本，但更加好用）
  • Local Historyopen in new window: 本地历史记录
  • Code Spell Checkeropen in new window: 单词拼写检查
  • joplin-vscode-pluginopen in new window: joplin 笔记集成

你可能注意到其中部分插件和 markdown 没什么关系，但它们对于编辑而言仍然是重要的，例如 Joplin 最近（v1.4.*）才实现的拼写检查功能已经存在现有的 VSCode 插件了。

# 替代品

joplin 最近开发了插件系统，也有许多开发者在为它编写 各种插件open in new window，而使用 joplin-vscode-plugin 则无法享受大部分插件带来的功能。然而，vscode 本身就有一个极其庞大的插件系统，以及繁多的功能。下面列出了一些 joplin plugin 在 vscode 的替代插件或功能

  • 特殊粘贴（例如 csv）open in new window：Plugin Markdown table from CSV/TSVopen in new window
  • vim 编辑：Plugin Vimopen in new window
  • 笔记大纲侧边栏open in new window：Outline viewopen in new window
  • markdown 表格格式化open in new window：Plugin Prettier - Code formatteropen in new window
  • 编辑器中的持久折叠open in new window：Foldingopen in new window
  • 模板功能open in new window：Snippetsopen in new window
  • diff 对比open in new window：Diff editor review paneopen in new window
  • math 模式：Markdown All in Oneopen in new window
  • rich markdownopen in new window：UNOTES - Markdown Notes WYSIWYGopen in new window
  • 编辑器布局open in new window：Tabsopen in new window
